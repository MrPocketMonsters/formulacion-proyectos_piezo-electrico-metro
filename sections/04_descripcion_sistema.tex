% Descripción del sistema propuesto

El sistema propuesto consiste en la instalación de losetas piezoeléctricas inteligentes en zonas estratégicas de las estaciones del metro, complementado con sistemas de almacenamiento y distribución de energía.

\subsection{Componentes del Sistema}

\subsubsection{Módulos de Captación}

\begin{itemize}
    \item Losetas piezoeléctricas de 50 × 50 cm
    \item Estructura de soporte y amortiguación
    \item Capa protectora antideslizante
    \item Sensores de carga integrados
\end{itemize}

\subsubsection{Sistema de Conversión y Acondicionamiento}

\begin{itemize}
    \item Rectificadores AC/DC
    \item Circuitos MPPT (Maximum Power Point Tracking)
    \item Reguladores de voltaje
    \item Protecciones eléctricas
\end{itemize}

\subsubsection{Sistema de Almacenamiento}

\begin{itemize}
    \item Banco de supercapacitores
    \item Baterías de ion-litio de respaldo
    \item Sistema de gestión de baterías (BMS)
\end{itemize}

\subsubsection{Sistema de Distribución}

\begin{itemize}
    \item Inversores DC/AC
    \item Tablero de distribución inteligente
    \item Medidores de energía
    \item Sistema de monitoreo en tiempo real
\end{itemize}
