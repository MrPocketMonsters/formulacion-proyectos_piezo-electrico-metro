% Alcance del proyecto

\subsection*{Alcance Geográfico}

El proyecto contempla la implementación piloto en [2-3] estaciones del sistema de metro, seleccionadas según criterios de alto flujo de pasajeros, disponibilidad de espacio y facilidades de integración con la infraestructura eléctrica existente.

\subsection*{Alcance Técnico}

El proyecto incluye:
\begin{itemize}
    \item Diseño e instalación de losetas piezoeléctricas en zonas de alto tráfico
    \item Sistema de captación y conversión de energía
    \item Sistema de almacenamiento de energía (baterías o supercapacitores)
    \item Sistema de distribución e integración con la red eléctrica existente
    \item Sistema de monitoreo y control del rendimiento energético
    \item Señalización informativa para usuarios
\end{itemize}

\subsection*{Alcance Económico}

El proyecto considera:
\begin{itemize}
    \item Inversión inicial en tecnología y instalación
    \item Costos de operación y mantenimiento
    \item Ahorros por generación de energía
    \item Análisis de viabilidad financiera a largo plazo
\end{itemize}

\subsection*{Lo que NO incluye el proyecto}

\begin{itemize}
    \item Modificaciones estructurales mayores en las estaciones
    \item Implementación simultánea en toda la red del metro
    \item Generación de energía a partir de otras fuentes renovables (solar, eólica, etc.)
    \item Sistemas de almacenamiento energético de gran escala para toda la estación
\end{itemize}
